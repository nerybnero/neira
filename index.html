Технология подбора банковских продуктов на базе искусственного интеллекта

Что такое Нейра?

Нейра - иновационный финтех-проект от команды ВЗО, который помогает подобрать лучший кредит для конкретного пользователя

Как это работает

Формируются профили поведения пользователей на основании статистических данных

Определяется значимость критериев для одобрения заявки на основании математической модели

Выявляется зависимость одобрения заявки в конкретных банках для разных сформированных профилей

Корректируется работа по мере накопления статистических данных

Нейра бесплатно показывает ваш кредитный рейтинг. На его основе она прогнозирует вероятность одобрения нового займа, а также обучает вас финансовой грамотности. Вы получаете персональные рекомендации по улучшению вашего благосостояния, учитесь управлять деньгами и планировать доходы.

О нас

Мы создаем автоматическую систему подбора кредитов в России, заботясь о наших пользователях

В ее создании принимает участие вся команда ВЗО, студенты и работники математического центра НГУ

Технологии создания Нейры

BigData

Массивы данных большого объема и разнообразия, анализируемые с помощью специальных компьютерных технологий. На их основе Нейра делает прогнозы

Машинное обучение

Массивы данных большого объема и разнообразия, анализируемые с помощью специальных компьютерных технологий. На их основе Нейра делает прогнозы

Поведение пользователя

Системы аналитики исследуют поведение пользователя на сайте. Эти данные Нейра использует для формирования персонального предложения

OpenAPI

Набор программных инструментов, через которые Нейра взаимодействует с системой банка и обменивается с ней данными

PostBack

С помощью этой технологии Нейра узнает решение банка по кредитной заявке — например, «одобрено» или «отказано». Такая информация помогает ей делать высокоточные прогнозы одобрения кредита

Кредитный рейтинг

Совокупность факторов, которая помогает Нейре точнее определить вероятность одобрения кредитной заявки. Он основан на кредитной истории пользователя
